# 1. O que é o Toqy

O Toqy é uma plataforma de **bio site profissional, cartão digital, QR Code e NFC para comércios locais**.

Ele será usado no ecossistema do Leonardo Marusso para:

1. Leonardo vender plaquinhas QR/NFC com bio site premium.
2. Alunos/revendedores criarem bio sites para clientes.
3. Clientes finais editarem o próprio bio site com uma chave de acesso.
4. Futuramente integrar com comunidade, curso, planos, créditos, afiliados, Supabase, uploads reais e pagamentos.

O Toqy **não é apenas um link na bio**. Ele deve parecer uma plataforma profissional para:

* Bio site editável.
* Cartão digital.
* QR Code.
* NFC.
* Plaquinhas físicas.
* Catálogo de produtos e serviços.
* Pix/Pix Hub.
* Wi-Fi com QR Code.
* Avaliação Google.
* WhatsApp.
* Agendamento.
* Salvar contato.
* Painel para alunos/admin.
* Área simples para cliente final editar.

# 2. Regra principal do produto

O sistema deve servir para qualquer negócio local, não apenas barbearia.

Segmentos obrigatórios:

## 3.2 Área do cliente final

Rota:

`/me`

Função:

Cliente final entra para editar o próprio bio site com chave de acesso.

Fluxo:

1. Cliente acessa `/me`.
2. Digita o link da página ou nome do bio site.
3. Digita a chave de acesso.
4. Se a chave estiver correta, entra na área de edição.
5. Vê apenas o próprio bio site.
6. Pode editar apenas informações permitidas.
7. Pode copiar link público.
8. Pode ver QR Code do próprio bio site.
9. Pode abrir a página pública.

Linguagem simples. Não usar termos técnicos como “slug”, “schema”, “HEX” ou “módulo” para o cliente.

Usar termos:

* Link da página
* Chave de acesso
* Cores
* Botões
* Imagem de fundo
* Catálogo
* Salvar alterações
